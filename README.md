# 腕立て200

腕立て200は、日々の腕立てメニューをランダムまたは手動選択で記録するためのシンプルな静的Webアプリです。

ブラウザだけで動作し、当日の記録は `localStorage` に保存されます。

## アプリ概要

以下の基本メニューをランダムに抽選、または手動で選択し、実施順と合計回数を記録します。

- 肩: 40回
- 大: 60回
- 中: 40回
- 小: 40回
- 零: 20回

合計で200回のメニューを完了することを目的としています。

## 主な機能

- 次に行う腕立て種目のランダム抽選
- 任意の種目を手動で選んで記録
- 当日の実施済み種目、残り種目、総回数の表示
- 実施順ログの表示
- 今日の記録のコピー
- 今日の記録のリセット
- 抽選設定の切り替え
  - 出た種目の重みを0%から100%まで選択できる
- 同じ種目が連続しないように自動調整
- 出た種目の重みは初期値0%
- 1件でも記録がある場合、出た種目の重みは変更不可
- 手動で選んだ種目も重み計算に反映
- 日付ごとの `localStorage` 保存
- ホーム画面追加用アイコンを設定済み

## ファイル構成

```text
.
├── index.html   # アプリのHTML構造
├── app.js       # 抽選、記録、localStorage保存などのアプリロジック
├── styles.css   # 画面デザイン
├── manifest.webmanifest # ホーム画面追加用のWebアプリ設定
└── icons/        # ホーム画面用アイコンとfavicon
```

## 起動方法

依存パッケージやビルド手順はありません。

ブラウザで `index.html` を直接開くことで動作します。

ローカルサーバーで確認する場合は、プロジェクト直下で以下を実行します。

```sh
python3 -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 動作確認方法

ブラウザでアプリを開き、以下を確認します。

1. 「次を引く」を押すと種目が表示される
2. 「手動で選ぶ」から任意の種目を押すと、その種目が記録される
3. 残り種目、実施済み種目、総回数が更新される
4. 順番ログに抽選結果または手動選択結果が追加される
5. 出た種目の重みを0%、10%、30%、50%、60%、80%、100%から選択できる
6. 初期値が0%になっている
7. 記録がある間は出た種目の重みを変更できない
8. 手動で選んだ種目も重み計算と連続回避に反映される
9. 同じ種目が連続しない
10. 「コピー」で今日の記録がクリップボードにコピーされる
11. リセットボタンで今日の記録を初期化できる
12. ページを再読み込みしても当日の記録と設定が保持される
13. iPhoneではSafariから「ホーム画面に追加」でアイコン表示を確認する
14. 既にホーム画面に追加済みの場合は、古いアイコンを削除して追加し直す必要がある場合がある
